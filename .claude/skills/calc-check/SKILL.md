# Calculator Verification
1. Read all input fields in the target HTML file
2. Trace each input through the JavaScript calculation logic
3. Verify every input is used in the final output/result
4. Check that cumulative/aggregate values multiply by all relevant factors
5. Report any inputs that are collected but not used in calculations

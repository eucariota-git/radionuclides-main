# NM Radionuclide Planner

Web application for nuclear medicine radionuclide properties, decay calculations, and external dose estimation for radiation workers.

**Live app:** https://eucariota-git.github.io/radionuclides-main/

Target users: medical physicists and radiation protection specialists.

---

## Features

### Properties (`index.html`)
- Database of 40 radionuclides used in nuclear medicine (PET, SPECT, therapy, brachytherapy)
  - 32 nuclides: published in Cornejo et al. (2015); operational values recalculated from ICRP 107 photon data with ICRU 57 conversion coefficients
  - 1 nuclide (Ga-68): addition beyond Cornejo, calculated from ICRP 107 photon data and ICRU 57 conversion coefficients
  - 6 therapy/imaging additions (2026-06): Zr-89, Cu-67, Tb-161, Ho-166, and the alpha-therapy chains **Ra-223+daughters** and **Ac-225+daughters** (constants include progeny in secular equilibrium; generated by `tools/add-therapy-nuclides.js`)
  - 1 nuclide (Y-90): pure β⁻ emitter; external dose from bremsstrahlung — container estimates following the methodology of Zanzonico et al. (1999); container values are not tabulated in that publication
  - Note: Some values differ from Cornejo due to updated nuclear data in ICRP 107; published Cornejo values are preserved in `cornejo_validation` field
- Controlled liquid effluent concentration limits calculated from IS-28 and ICRP 119 adult ingestion coefficients
- Filterable by category and modality

### Decay calculator (`decay.html`)
- A(t) = A₀ · e^(−λt) with interactive Chart.js v4.4.0 curve (loaded locally, no CDN dependency)
- Supports MBq, GBq, mCi, Ci
- Time to target activity

### Dose estimator (`dose.html`)
- **Dose rate at user-selected distance** (primary output) with 1 m reference column
- **H\*(10)** ambient dose equivalent for whole body (d ≥ 25 cm)
- **H'(0.07)** directional dose equivalent for extremities (d < 25 cm); used when available for low-energy emitters, otherwise H*(10) is conservatively applied
- **Y-90 bremsstrahlung container estimates** (methodology after Zanzonico et al. 1999; container values not tabulated in that publication) with interactive container selection (PMMA, Pb, W, none); applies H'(0.07) at d < 25 cm where applicable; Pb warning and dose consolidation via `PHYSICS.Y90_CONTAINERS`
- Shielding: Lead and concrete (normal-weight and light-weight)
- **Archer broad-beam method** (Monte Carlo, Oumano et al. 2025) for Tc-99m, F-18, I-131, Lu-177
- **Spectrum-weighted narrow beam** (dose-weighted sum over ICRP 107 photon lines, NIST XCOM) for all other nuclides — accounts for beam hardening of multi-line emitters; single-line fallback for custom nuclides
- **📄 Report / PDF** — printable calculation report with timestamp, database version, inputs, results, method, data sources and disclaimer (also in the Decay calculator)
- Cumulative dose with radioactive decay integration
- **Annual dose limit comparison** (RD 1029/2022 / EURATOM 2013/59):
  - Effective dose: 20 mSv/y worker, 1 mSv/y public
  - Extremities: 500 mSv/y
  - **Lens of eye:** Both conditions must be satisfied: 50 mSv/y max annual AND 100 mSv over any consecutive 5-year period (current regime since 22 Jun 2024)
  - Crystal lens calculation with both limits displayed
- CSV export includes dose rates at both selected distance and 1 m reference

### Custom nuclide (`custom.html`)
- Upload CSV in IAEA LiveChart format
- Calculates Γ^H\*(10) and Γ^H'(0.07) using Cornejo et al. methodology (ICRU 57 / ICRP 74)
- Filter criteria: photons (G/X), E ≥ 20 keV, yield ≥ 0.01%

### Extended database — ICRP 107 (`index.html` + `validate-icrp107.html`)
- Main database contains 40 curated radionuclides. For any other nuclide, the Properties page offers fallback search in **ICRP Publication 107** (1252 nuclides)
- **"Search in ICRP 107" button** appears when a nuclide is not found in the main database
- Automatically calculates Γ^H\*(10) and Γ^H'(0.07) from ICRP 107 photon emissions, including annihilation quanta where present (E ≥ 20 keV, yield ≥ 0.01%)
- Displays photon spectrum (energy, yield, type) with **max photon energy** for H'(0.07) assessment
- **Validation tool** (`validate-icrp107.html`) — internal compliance tool:
  - Tests 20 reference nuclides (7 original + 8 Phase 2 + 5 Phase 3, including I-125, Cu-64, Re-188, Tm-170, Au-198, Yb-169)
  - Validates: Z extraction, half-life parsing, photon count filtering, dose constant calculation
  - Color-coded status: ✓ Pass (< 5% deviation) | ⚠ Warning (5–15%) | ✗ Fail (> 15%)
  - All tests currently passing
- Parser metadata: SHA256 hashes of source ICRP-107 files stored in JSON for audit trail
- See `references/icrp107/README.md` for detailed documentation on ICRP 107 integration

## Physics

### Dose rate constants
Γ [μSv·h⁻¹·GBq⁻¹·m²] = K · Σᵢ (nᵢ · h(Eᵢ))

where K = 28.648, nᵢ = photon yield per decay, h(Eᵢ) = fluence-to-dose conversion coefficient (ICRP 74).

### Shielding — Archer equation (broad beam)
T(x) = [(1 + β/α) · e^(αγx) − β/α]^(−1/γ)

Parameters from Monte Carlo simulations (GATE/MCNP6) for Tc-99m, F-18, I-131, Lu-177 in lead, normal-weight concrete and light-weight concrete.
Source: Oumano et al., *J Appl Clin Med Phys*, 2025. doi:10.1002/acm2.70084

### Shielding — spectrum-weighted narrow beam with build-up (all other nuclides)
T(x) = Σ wᵢ · min(1, B(Eᵢ, μᵢx) · e^(−μᵢx))

wᵢ = relative H\*(10) dose contribution of each photon line (ICRP 107 yields × ICRU 57, normalized; stored as `shielding_spectrum` in `data/nuclides.json`, generated by `tools/add-shielding-spectra.js`). This captures **beam hardening** of multi-line emitters — a single-line exponential can underestimate the transmitted dose by orders of magnitude behind thick shields.

B(E, mfp) = **exposure buildup factor** for a point isotropic source, interpolated (log-B; linear in mfp, log-log in E) from the ANSI/ANS-6.4.3 standard reference data — Trubey, *NUREG/CR-5740 / ORNL/RSIC-49/R1* (1991), Table 3, Lead (with K-edge split) and Concrete (shared by both density variants — B depends on mean free paths, not density). The min(1, ·) clamp removes the infinite-medium artifact T > 1 at low energies. Infinite-medium buildup slightly **overestimates** transmitted dose for finite barriers (conservative). Validity 0–40 mfp (clamped beyond). Cross-validated against the Archer Monte Carlo fits for Tc-99m/I-131: agreement within a factor ~0.5–2 at 2–10 HVL, where the pure exponential was wrong by orders of magnitude (test 12e).

μ from NIST XCOM (includes coherent scattering — slightly overestimates the mfp count, hence B: conservative, second-order); lead K-edge at 88 keV handled explicitly. Single-line narrow beam (also with build-up) remains the fallback for custom nuclides without a stored spectrum.

## Data Integrity & Traceability

All generated JSON files include metadata for audit trail:

```json
{
  "notes": {
    "generated_at": "2026-05-17T19:27:19.466Z",
    "generated_by_script": "tools/parse-icrp107.js",
    "icrp107_ndx_sha256": "6aa2f8d762c9d2df0b4848b9ed13979a1f5b71f2a7c1889e28b5512b46cbfd80",
    "parser_version": "2.0 (fixed-width columns)"
  }
}
```

- **Parser version 2.0:** FORTRAN fixed-width column extraction (not space-splitting)
- **Z values:** Derived from nuclide symbol, validated against ICRP-107
- **Hashes:** Allow verification that source files (NDX/RAD/JSON) have not been altered
- **Timestamps:** Track when data was last regenerated

## Recent Improvements & Bug Fixes (2026-06)

### Round 3 (v1.2) — buildup factors
- **Exposure build-up added to all narrow-beam shielding** (spectral and single-line): T = Σ wᵢ·min(1, B·e^(−μx)) with ANSI/ANS-6.4.3 point-isotropic exposure buildup factors (NUREG/CR-5740 Table 3; public-domain US-government report, values transcribed programmatically and verified visually against the scan). Closes the last documented non-conservative gap — without B, transmitted dose behind thick shields was underestimated (concrete: factor 2–10+). HVL/TVL and required-thickness now solved numerically. Cross-validated against the Archer fits (ratios 0.5–2.1 at 2–10 HVL). Archer nuclides unchanged. 18 new tests (math suite: 54).

### Round 2 (data v1.1)
- **Spectrum-weighted narrow-beam shielding:** single-energy attenuation underestimated transmitted dose for multi-line emitters (Ga-67 behind 5 mm Pb: ×10¹² error; I-123: ×430). All non-Archer nuclides now use T(x) = Σ wᵢ·e^(−μ(Eᵢ)x) over their dose-weighted ICRP 107 lines (`shielding_spectrum`, 39 nuclides incl. the Mo-99+Tc-99m and Cs-137+Ba-137m equilibria and the Ra-223/Ac-225 chains). Archer nuclides are unaffected (their parameters already fit the full spectrum). HVL/TVL and required-thickness solve the spectral equation numerically.
- **Chain max photon energy fixed:** Ra-223 632 → 1196.3 keV, Ac-225 526.1 → 1567.1 keV (progeny lines — Pb-211 832 keV, Tl-209 1567 keV — were missed by the original generation).
- **Printable calculation reports** (`js/report.js`): Dose and Decay calculators generate a formal report (timestamp, database version, nuclide validation status, inputs, results, method, sources, disclaimer, sign-off line) via the 📄 Report / PDF button.
- **PWA:** `manifest.json`, `favicon.svg` and a cache-first service worker (`sw.js`) — installable on Android/desktop, fully offline once visited.
- **Clearance verification completed:** Lu-177 = 100 kBq/kg (plus Sm-153, Ho-166, Tm-170, Yb-175) visually confirmed against the official RD 1217/2024 PDF (p. 96).
- Decay calculator shows a secular-equilibrium note for chain entries (Ra-223, Ac-225).

### Critical
- **ICRP 107 RAD parser fix:** the block-header regex did not accept half-lives in scientific notation ("2.00E+5y"), so the photon blocks of **77 nuclides** were silently absorbed into the previous nuclide (e.g. Re-186 absorbed Re-186m, inflating its Γ by +101%; Ho-166 absorbed Ho-166m; the Ra-223/Ac-225 chain daughters had no photons at all). All 1252 RAD blocks now match their declared emission counts; Re-186 Γ^H*(10) = 3.86 matches the published Cornejo value exactly.
- **h'(0.07) column rebuilt:** the previous fluence-to-H'(0.07) column failed a kerma-ratio consistency test (implied H'(0.07)/Ka up to 3.7 — physically impossible). Rebuilt as (Ka/Φ)·hK'(0.07,0°) from ICRP 74 / PTB-Dos-34 (≤100 keV exact) plus anchors derived from Cornejo's published per-line contributions; ≥500 keV uses the kerma approximation h'(0.07)≈h*(10). Reproduces all four published Cornejo Γ^H'(0.07) values within 1.3%. The nuclides selecting H'(0.07) for extremities now coincide exactly with the four published in parentheses by Cornejo (Pd-103, In-111, I-125, Xe-133).
- **Clearance levels verified** against RD 1217/2024 Anexo IV Tabla A.1 (BOE-A-2024-25205 consolidated HTML): 26 values confirmed; Ga-67 and Yb-169 are NOT listed in Tabla A.1 and now carry null.

### New
- **6 therapy/imaging nuclides added** (34 → 40): Zr-89, Cu-67, Tb-161, Ho-166, Ra-223+daughters, Ac-225+daughters. Chain constants include progeny in secular equilibrium; Ra-223+progeny agrees within 6% with the published conservative constant (~50 μSv·h⁻¹·GBq⁻¹·m²). Generated by `tools/add-therapy-nuclides.js`.
- **Ingestion e(g) and liquid effluent limits** for the 6 new nuclides taken from ICRP Publication 119 Annex F, Table F.1 (adult), via `tools/add-ingestion-coeffs.js`. The same run cross-checked the 30 pre-existing adult e(g) values against Table F.1 — all match exactly; a new test guards them.

## Compatibility

Works on desktop and mobile browsers. Loads from `file://`, `content://` (Android) and HTTP/HTTPS without a server — all dependencies are local:
- Curated and ICRP 107 data embedded as inline JS variables (`data/nuclides-data.js`, `data/icrp107-data.js`)
- Chart.js v4.4.0 bundled locally (`js/chart.umd.min.js`, no CDN dependency)
